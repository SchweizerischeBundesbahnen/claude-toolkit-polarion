# CLAUDE.md

- `plugins/core/CLAUDE.md` is redistributed to end-users when they install the core plugin — changes there affect users, not just this repo. Must stay aligned with the `/claude-md` skill principles.
- `plugins/core/CLAUDE.md` was intentionally slimmed to ~28 lines (from 266). Do not re-add generic SE principles — research shows long context files degrade agent performance and cost 20-23% more tokens.
- As AI models improve, remove entries that models no longer need to be told. Overprescriptive guidance ages into a liability.
- Conventional commits are enforced (`feat:`, `fix:`, `chore:`, etc.) — wrong format blocks the commit.
